# Django Portfolio Website

This is my personal portfolio website built using Django.

Technologies Used:
- Python
- Django
- HTML
- CSS
- Bootstrap

Features:
- Personal portfolio sections
- Projects showcase
- Contact form

How to run the project:

1. Clone the repository
git clone https://github.com/Swati-2805/portfolio_django.git

2. Go to project folder
cd portfolio_django

3. Run server
python manage.py runserver

Then open:
http://127.0.0.1:8000/
